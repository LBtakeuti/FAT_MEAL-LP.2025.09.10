# タスク完了時のチェックリスト

## 必須確認事項

### 1. リント実行
```bash
npm run lint
```
- 警告・エラーを確認
- 必要に応じて修正

### 2. ビルド確認
```bash
npm run build
```
- ビルドエラーがないことを確認
- TypeScriptの型エラーがないことを確認

### 3. 動作確認
```bash
npm run dev
```
- 変更した機能が正しく動作するか確認
- コンソールエラーがないか確認

### 4. レスポンシブ確認
- モバイル表示（`md:`以下）
- デスクトップ表示（`lg:`以上）

## 推奨確認事項

### コード品質
- 不要なconsole.logを削除
- 未使用のimportを削除
- コメントが適切か確認

### セキュリティ
- 機密情報がハードコードされていないか
- APIエンドポイントの認証が適切か

### パフォーマンス
- 画像の最適化
- 不要な再レンダリングがないか

## デプロイ前
- `.env.local`の本番値確認
- Vercel環境変数の設定確認
